# Telegram Bot Messages

## Tone

- коротко;
- по-человечески;
- без канцелярита;
- без внутреннего технического мусора.

## Speed / Health explanation

- **Speed** — более быстрый и простой режим. Используй его как основной, если всё работает стабильно.
- **Health** — более живучий режим для сложных условий и блокировок. Переключайся на него, если Speed работает нестабильно.

## Success messages

- Код активирован ✅
- Устройство удалено ✅
- Конфиг готов ✅

## Error messages

- Невалидный или неактивный код.
- Доступ неактивен: пополните баланс / активируйте invite code.
- Конфиг ещё готовится, попробуйте позже.

## Security policy for text

- не показывать пользователю внутренние UUID;
- не показывать полные invite codes в истории;
- не логировать и не печатать полные ключи в служебных логах.
