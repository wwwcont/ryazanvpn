# Telegram Bot Admin Flows

## Admin access

Admin определяется по списку `admin_ids` (telegram ID).

## Admin menu

- Пользователи
- Баланс пользователя
- Invite codes
- Ноды
- Статусы нод
- Перевыпуск доступа
- Заблокировать
- Разблокировать

## Пользователи

- Поиск по `@username` или `telegram id`.
- Карточка: статус, доступ, устройство, трафик.

## Баланс пользователя

- Команда в формате: `@username +100 причина` или `@username -50 причина`.
- Бот логирует запрос и даёт безопасный ответ.
- Финансовое применение выполняется через backend API/ledger.

## Invite codes

- Создать один код.
- Создать пакет кодов.
- Посмотреть последние коды.

## Ноды

- Список нод, статус, capacity/load, last heartbeat (когда доступно).

## Блокировка/разблокировка

- Блокировка в UX запускает отзыв доступов.
- Разблокировка в UX подтверждается и требует перевыпуска ключей при необходимости.
